terraform-homework-help-infrastructure/

├── variables.tf              # Variable definitions for Terraform

├── templates/                # Folder containing template files
│   └── server.js.tpl         # Template for the backend `server.js` file
├── outputs.tf                # Outputs of the infrastructure (e.g., EC2 public IP, S3 bucket name)
└── README.md                 # Documentation on how to deploy and use the infrastructure
